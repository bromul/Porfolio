Mapping Earthquakes

Using USGS data on earthquakes in the past 7 days, this project creates a map that plots each earthquake event along with information on its magnitude and location. Additionally, with data from this Github repository we were able to plot the boundaries of the tectonic plates in relation to earthquake events. Finally, using more USGS data on major earthquakes in the past 7 days (earthquakes whose magnitudes are 4.5+), we added a layer of only these events.
